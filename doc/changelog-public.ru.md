# Публичный changelog (маркетинговый)

## 2026-03 — MAX-бот запущен в продакшн

- **Второй мессенджер:** StaffProBot теперь работает в Telegram и MAX — единая бизнес-логика, два канала доставки
- **Смены и объекты:** открытие/закрытие смен, управление объектами, планирование, задачи — всё доступно через оба мессенджера
- **Привязка аккаунтов:** пользователи связывают MAX с аккаунтом StaffProBot через одноразовый код в личном кабинете
- **Уведомления:** персональные и групповые уведомления доходят в Telegram и/или MAX по настройкам владельца
- **Чаты отчётов:** владельцы настраивают чаты Telegram и MAX для объектов и организаций
- **Веб-интерфейс:** карточки сотрудников, договоры, формы менеджеров обновлены для работы с обоими мессенджерами

## 2026-03 — Подготовка к MAX-боту (Фаза 1)

- **Модель данных:** при регистрации создаётся запись в `messenger_accounts`; report chat читается из `notification_targets` (fallback на legacy поля)
- **Веб:** исправлено разделение user_id и telegram_id в owner-интерфейсе; при редактировании объекта записывается target для Telegram
- **Celery/бот:** все места получения report chat переведены на единый сервис notification_targets
